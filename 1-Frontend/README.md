# GBV Frontend Coding Task
Create a simple frontend for users to connect wallet and display the balance of the wallet.  

1. Create a `Connect` button
2. When user clicks `Connect`, pops out Metamask extension
3. After user connected wallet, display the network connected and the wallet balance of the user on the frontend

To obtaina a clearer idea, you may want to check out [this sample](https://web3-react-mu.vercel.app/), but our task should be simpler.

To complete the task, you may want to check out [`NoahZinsmeister/web3-react`](https://github.com/NoahZinsmeister/web3-react) & [`tmm/wagmi`](https://github.com/tmm/wagmi).

Please submit the Github repository link to the completed task with your application.
